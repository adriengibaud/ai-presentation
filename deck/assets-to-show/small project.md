This project is a web app meant to allow user to follow their finances. 
Upload their transaction, organize them, budgeting ( envelope mode ), FIRE tracker. All meant for swiss peoples.

## Stack
It uses tanstack start with
- ORPC as an API layer
- Better auth for authentication
- Tanstack query
- Tanstack form
- Shadcn
- Drizzle
- Postgres

## Validation

You should run formating & linting after each step. You should fix all error AND warnings.

All test must pass. We have multi layer tests. All should pass to consider your work done.

## Others

Before creating a new UI component, you should ALWAYS check that it does not already exists. If not, check shadcn lib and use one from here. If you really ned to create a new one ask before.
